# オリジナルポモドーロタイマー - 会話メモ

## プロジェクト概要
ユーザー独自のタイミング・ルールに基づいたポモドーロタイマーを作成する。

## 技術方針（段階的リリース戦略）

### Phase 1：PWA（最優先）
- ブラウザベースでPC・スマホ対応
- Service Workerによるオフライン完全対応
- ホーム画面追加でアプリライクな操作感

### Phase 2：Galaxy Watch（Wear OS）
- Kotlin + Android Studio で開発
- 現在のWindows環境で開発可能（Macは不要）
- PWAで確立したロジックを移植

### Phase 3：Apple Watch
- Swift/SwiftUI + Xcode で開発
- Mac環境が必要（現在未所持の場合は後回し）
- Apple Developer アカウント（$99/年）が必要
